RESTful API для магазина мерча. 

* cmd/app — точка входа.
* internal/handler — HTTP-слой.
* internal/service — слой бизнес-логики.
* internal/repository — слой работы с данными.
* internal/model — доменные сущности и DTO.
* pkg/ — переиспользуемые пакеты.

Реализовано
 * Интеграция с PostgreSQL.
 * Регистрация и авторизация пользователей.
 * Генерация и валидация JWT-токенов.
 * Контейнеризация приложения.

План разработки
 * Покупка предметов.
 * Переводы монет.
 * Логи.
 * Тесты.

Запуск
1. Создайте файл .env 
2. Запустите сборку и старт контейнеров:
   docker-compose up -d --build

API будет доступно по адресу: http://localhost:8080
